## 运行环境

- 后端：JDK 1.8 或以上版本
- 前端：现代浏览器
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 安装和配置 MySQL 数据库。
2. 部署后端服务，配置相应的数据库连接信息。
3. 部署前端项目，确保与后端服务网络通信正常。
